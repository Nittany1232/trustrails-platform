The widget is fully themeable using CSS variables. Apply these variables to your widget container or globally:

/* Example: Corporate Green Theme */ .trustrails-widget { --trustrails-primary-color: #059669; --trustrails-primary-hover: #047857; --trustrails-primary-bg: #ecfdf5; --trustrails-font-family: 'Your Brand Font', sans-serif; --trustrails-border-radius: 4px; }
